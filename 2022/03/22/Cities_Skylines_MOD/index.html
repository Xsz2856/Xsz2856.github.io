Loading Screen Mod temporary fix for Airports DLC

创意工坊链接：点击进入

作者：Klyte45、algernon
本地化：支持中文
是否需要DLC：否
前置模组：无

作用

  • 优化地图加载过程，查看资产情况以及内存占用情况。加快加载游戏速度。

  • 通过导出的资产报告查看“丢失”或“重复”的资产，并能通过超链接快速的进行订阅或取消订阅。

Loading Order Mod

创意工坊链接：点击进入

作者：kian.zarrin、Chamëleon TBN、LeonardMT、LemonsterOG
本地化：不支持中文是否需要DLC：否前置模组：Patch Loader Mod、Harmony 2.2-0 (Mod Dependency)

作用：

  • 精简主菜单

  • 调整资产加载顺序，将Harmony 2.2-0 (Mod Dependency)模组放在首位加载，加快存档载入速度。

  • 加载顺序工具：此模组包含一个名为LoadOrderTool.exe的程序，通常它的位置为[游戏磁盘:\SteamLibrary\steamapps\workshop\content\255710\2448824112\Tool]{.kbd}或者[游戏磁盘:\Steam\steamapps\workshop\content\255710\2448824112\Tool\LoadOrderTool.exe]{.kbd}。LoadOrderTool是一个非常强大的内容管理器，可让您创建/导出/导入模组和资产的不同预设。此外，它还添加了您可以选择的各种启动参数。

  • LoadOrderTool.exe需要[.NET Desktop Runtime 3.1+]{.kbd}。Windows 64位的电脑可以点击此处下载。其他操作系统的版本可在此处下载。安装[.NET Desktop Runtime 3.1+]{.kbd .red}后请重启电脑！

创意工坊链接：点击进入

作者：Dendraspis
本地化：不支持中文是否需要DLC：否前置模组：无

作用：您可以使用此模组重新定位工具栏面板并调整其大小，以便更好、更快地使用。工具栏面板表现为普通或扩展，高度和宽度可调。此外，还有一些功能可以自定义工具栏。

  • 可隐藏的扩展按钮，其中包含有关工具栏面板大小和比例的信息，以及用于切换模式和其他功能的实时时钟（可选）

Prop Anarchy 0.7.5

创意工坊链接：点击进入

作者：Quistar、Chamëleon TBN
本地化：支持中文是否需要DLC：否前置模组：Move It、Extended Managers Library 1.0.3、Harmony 2.2-0 (Mod Dependency)

作用：

此模组整合了许多主流模组的功能

  • 道具限制可以通过选项菜单中的滑块进行调整

  • 允许您将道具放置在任何地方（功能同Prop & Tree Anarchy）

  • 允许您调整道具高度，并捕捉到建筑物/网络

  • 允许您使用大于8x8的贴花资源（与原始mod相比，加载时间更长）

  • 摆件吸附，可以在建筑物上吸附摆件，更精准放置道具（功能同Prop Snapping）

  • 可以像画道路一样画出道具和树木线条（功能同Prop Line Tool）

  • 附加着色器 - 移植了此自定义效果，该效果在许多资源中使用。现在，它的加载速度要快得多，并且在玩游戏期间不会占用系统资源。（功能同Additive Shader）

  • 道具大小调整—使用Move it选中道具按键盘上的[。]{.kbd}和[，]{.kbd}来放大和缩小道具。

此模组存在许多不兼容的模组，但也给出了代替方案。请仔细阅读创意工坊页面的介绍{.dot}。

Tree Anarchy 1.3.5

创意工坊链接：点击进入

作者：Quistar、Chamëleon TBN
本地化：支持中文是否需要DLC：否前置模组：Move It、Harmony 2.2-0 (Mod Dependency)

作用：

此模组整合了许多主流模组的功能

  • 你可以种植超过262144棵树，限制通过滑块进行调整（功能同Unlimited Trees Mod v1.12和Unlimited Trees: Revisited）

  • 加载旧的无限树模组保存的游戏/地图

  • 允许将树木捕捉到建筑物和高架道路 - 就像在资源编辑器中一样（功能同Tree Snapping）

  • 调整树的大小超出游戏限制

  • 随机树旋转（功能同Random Tree Rotation）

  • 允许您锁定和解锁林业资源，因此添加和移除树木不会改变地面资源（林业、沃土、矿石和石油）或地面纹理（外观）（功能同Lock Forestry）

  • 提高树 LOD 的质量（功能同Tree LOD Fix）

此模组存在许多不兼容的模组，但也给出了代替方案。请仔细阅读创意工坊页面的介绍{.dot}。

More PathUnits (Beta Testing)

创意工坊链接：点击进入

作者：algernon、Chamëleon TBN
本地化：该模组无界面是否需要DLC：否前置模组：Harmony 2.2-0 (Mod Dependency)

作用：使游戏中车辆的路径数量增加一倍，从262144增加到524288

  • 什么是PathUnits？

    PathUnits是游戏的内部数据结构，用于执行寻路（从A到B）。将一辆车（或公民）从其起点到目的地将需要许多PathUnits（取决于所需路线的长度）。

  • 当游戏路径单位达到上限时会发生什么？

    不幸的是，与其他一些限制不同，当您达到PathUnit上限时，游戏不会通知您。缺少PathUnits意味着游戏无法将车辆或公民引导到目的地;公共交通停止工作，城市服务（包括消防车！）无法正常生成，货物交付停止发生。当你注意到的时候，你可能已经有多座废弃的建筑物。

注意！此模组会影响存档游戏！如果你在启用此模组的情况下保存游戏，然后禁用或取消订阅该模组，你将无法加载你的存档！

More CitizenUnits 1.0

创意工坊链接：点击进入

作者：algernon
本地化：该模组无界面是否需要DLC：否前置模组：Harmony 2.2-0 (Mod Dependency)

作用：使游戏中市民的路径数量增加一倍，从524288增加到1048576

  • 什么是Citizen Units？

    • CitizenUnits（不要与Citizens混淆）是游戏的内部数据结构，用于将公民"分配"到建筑物，车辆等。每个家庭，工作场所每五个可用工作，商业建筑中每五个可用客户名额，以及每五个娱乐场所（例如公园/独特建筑/公园区域/体育场/等），学校/学院/大学/图书馆/等每五个学生名额，诊所/医院/健身房/等每五个患者都需要一个公民单位。 公共交通车辆中的每五个车位等等。
    • 无论这些地方是否被占用，都会分配这些公民单位;例如，即使没有人真正参观该公园或上学或乘坐该公共汽车旅行，公民单位仍然必须创建并准备好接待公民。
    • 从中可以看出，游戏中CitizenUnit的数量可以很容易地超过你的实际人口，有时是许多倍数 - 但游戏的硬编码限制为524287 CitzenUnits。一旦达到此限制，它就不能再分配了。这只是一个基本的游戏限制，比如节点限制或段限制或建筑限制或路径限制或区域块限制或车辆限制或公民本能限制等。

  • 当游戏Citizen Unit达到上限时会发生什么？

    不幸的是，与其他一些限制不同，当您达到CitizenUnit限制时，游戏不会通知您。它只是继续前进，在后台悄悄地失败。通常，人们注意到的第一个现象是0/0家庭的住宅楼。其他现象包括工作场所的就业机会少于应有的工作，没有访客的商业和/或娱乐建筑，无法搭载乘客的公共交通车辆，无法容纳学生的教育建筑等。

注意！此模组会影响存档游戏！如果你在启用此模组的情况下保存游戏，然后禁用或取消订阅该模组，你将无法加载你的存档！
